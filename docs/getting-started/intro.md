## 1. 入门指南

欢迎使用 ko-blog 文档！

### 1.1. 开始

#### 1.1.1. 安装依赖

```bash
npm install
```

#### 1.1.2. 启动开发服务器

```bash
npm start
```

#### 1.1.3. 构建生产版本

```bash
npm run build
```

### 1.2. 目录结构说明

| 目录 | 说明 |
|------|------|
| `/blog` | 博客文章 |
| `/docs` | 文档页面 |
| `/src/pages` | 自定义页面 |
