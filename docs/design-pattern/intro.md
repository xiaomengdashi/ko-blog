# 设计模式

设计模式是软件开发中常见问题的经典解决方案。

## 目录

### 创建型模式
- 单例模式 (Singleton)
- 工厂模式 (Factory)
- 抽象工厂 (Abstract Factory)
- 建造者模式 (Builder)
- 原型模式 (Prototype)

### 结构型模式
- 适配器模式 (Adapter)
- 装饰器模式 (Decorator)
- 代理模式 (Proxy)
- 外观模式 (Facade)
- 组合模式 (Composite)

### 行为型模式
- 观察者模式 (Observer)
- 策略模式 (Strategy)
- 命令模式 (Command)
- 迭代器模式 (Iterator)
- 状态模式 (State)

### 设计原则
- 单一职责原则 (SRP)
- 开闭原则 (OCP)
- 里氏替换原则 (LSP)
- 接口隔离原则 (ISP)
- 依赖倒置原则 (DIP)
