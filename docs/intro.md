# 欢迎使用 ko-blog

这是一个基于 [Docusaurus 3](https://docusaurus.io/) 构建的技术博客。

## 功能特点

- 📝 **博客** - 支持Markdown/MDX格式的博客文章
- 📚 **文档** - 组织良好的技术文档系统
- 🔍 **搜索** - 内置全文搜索
- 🌙 **暗色模式** - 自动适应系统主题
- 🌐 **国际化** - 支持多语言
- 🚀 **SEO友好** - 优化的搜索引擎支持
