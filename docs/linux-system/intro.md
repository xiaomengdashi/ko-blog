# Linux 系统

Linux 是服务器和嵌入式领域最重要的操作系统。

## 目录

### 基础命令
- 文件操作 (ls, cd, cp, mv, rm)
- 文本处理 (cat, grep, sed, awk)
- 权限管理 (chmod, chown)
- 进程管理 (ps, top, kill)

### Shell 编程
- 变量与数据类型
- 条件判断与循环
- 函数
- 正则表达式

### 系统编程
- 文件 I/O
- 进程与线程
- 信号处理
- 内存管理

### 系统管理
- 用户管理
- 磁盘管理
- 网络配置
- 日志管理

### 性能优化
- CPU 分析
- 内存分析
- I/O 分析
- 系统调优
